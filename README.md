# Counter App

## Project Description

**Counter App** is a simple and interactive web application built using **HTML, CSS, and JavaScript**. It allows users to increase, decrease, and reset a counter with a clean, responsive, and user-friendly interface. This project demonstrates the fundamentals of DOM manipulation, event handling, and dynamic UI updates using vanilla JavaScript.

The application is ideal for beginners learning JavaScript and serves as a practical example of how user interactions can update webpage content in real time.
